# vrmnx-toruo

VRMNX用Python拡張の「撮る夫くん」は，VRMNXビュワーのフライスルーカメラの機能をアップグレードします。

## 主要な機能

- GUI操作のFOV（ズーム）
- 一眼レフカメラの焦点距離をシミュレートした被写界深度の操作
- 手ブレ風エフェクト
- 車両追尾機能
