組み込み方法 ==========
レイアウトスクリプトに
import toruo
を追加し，イベントハンドラの中で

def vrmevent(obj, ev, param):
    toruo.activate(obj,ev,param)
    if ev == 'init':
        #以下省略

のように toruo.activate() を呼び出してください。


操作方法 ==========
キーボード -----
[w][e][r] ズームイン
[s][d][f] ズームアウト
[p] GUIウィンドウを表示・消去

使用上の条件 ==========
撮る夫くんを使用したスクリーンショットまたは動画の撮影および公開はこれを自由とします。
レイアウトファイルの配布に際してtoruo.pyを同梱する形での再配布はこれを自由とします。
ただし，撮る夫くんは今後のアップデートによって互換性を失う可能性があります。
